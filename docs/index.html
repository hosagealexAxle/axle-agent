Personal Use · Desktop App
Axle

A desktop operator console for managing an Etsy shop — budget tracking, SEO planning, listing analytics, and shop automation in one place.

Features

💰 Budget Enforcement

Real-time budget tracking with monthly and daily caps across API, SEO, and Ads categories. Color-coded progress bars warn before overspending.

📊 Shop Analytics

Dashboard displays visits, orders, revenue, and conversion rate. Track listing-level performance metrics over time.

🔎 SEO Planning

Plan and track SEO improvements for listings. Chat-based workflow helps optimize titles, tags, and descriptions.

🤖 AI Chat Assistant

Built-in conversational assistant with persistent memory. Remembers your shop strategy, preferences, and key facts across sessions.

📈 Listing Management

View and manage listing performance data. Track individual listing visits, orders, revenue, and conversion rates.

🔒 Policy Controls

Configurable safety guardrails — ROI minimums, confidence thresholds, approval requirements, and a kill switch to halt all automated actions.

Etsy API Integration

Axle connects to the Etsy Open API v3 to automate shop data retrieval for a single shop (MrsMillennialDesigns). This is a personal-use application — not a commercial multi-seller platform.

The app uses OAuth 2.0 with PKCE for secure authentication and requests only the scopes necessary for shop management:

shops_r listings_r listings_w transactions_r profile_r

How It Works

1

Connect Your Shop

Authenticate with Etsy via OAuth 2.0. Axle receives read/write access to your shop data only.

2

Data Syncs Automatically

Shop stats, listing metrics, and order data flow into the dashboard. No more manual CSV exports.

3

Plan & Execute

Use the chat assistant to plan SEO changes, manage budgets, and make data-driven decisions — all with guardrails to prevent overspending.
