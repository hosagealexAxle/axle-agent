Privacy Policy

Axle by MrsMillennial Designs — Last updated: February 23, 2026

Overview

Axle is a personal-use desktop application built to manage a single Etsy shop (MrsMillennialDesigns). It is not a commercial product and is not offered to third parties. This privacy policy describes how the application handles data from the Etsy and Pinterest APIs.

Data We Collect

When you authenticate Axle with your Etsy or Pinterest account, the app accesses:

  • Etsy: Shop profile, listings, listing statistics, orders, and revenue data for the connected shop.
  • Pinterest: User profile, boards, and pins for the connected account.

No data is collected from any user other than the shop owner who authenticates the app.

How Data Is Stored

  • All data is stored locally on the shop owner's computer in a SQLite database.
  • There are no cloud servers, no external databases, and no third-party analytics.
  • OAuth tokens are stored locally and are never transmitted to any server other than the official Etsy and Pinterest API endpoints.

How Data Is Used

Data retrieved from Etsy and Pinterest is used exclusively to:

  • Display shop analytics and listing performance in the local dashboard.
  • Help the shop owner plan SEO improvements and manage listings.
  • Create and manage Pinterest pins to promote shop listings.
  • Track budgets and spending across shop operations.

Data is never sold, shared, or disclosed to any third party.

Third-Party Services

Axle connects to the following APIs solely for shop management:

  • Etsy Open API v3 — to read and manage shop data.
  • Pinterest API v5 — to create pins and manage boards.
  • Anthropic (Claude) API — to power the AI chat assistant. Chat messages are sent to Anthropic's API for processing. Anthropic's privacy policy applies to that data.
  • OpenAI API — used as a fallback AI provider. OpenAI's privacy policy applies to that data.

Data Retention

Data persists in the local database as long as the application is installed. Uninstalling the app and deleting its data folder removes all stored data. There is no remote backup or cloud sync.

Data Sharing

Axle does not:

  • Share data with advertisers or data brokers.
  • Collect data from users other than the authenticated shop owner.
  • Use tracking pixels, cookies, or analytics services.
  • Transmit data to any server other than the official API endpoints listed above.

Security

  • OAuth 2.0 with PKCE is used for Etsy authentication.
  • OAuth 2.0 is used for Pinterest authentication.
  • All API communication uses HTTPS/TLS encryption.
  • Remote access (iPhone) is secured via Tailscale's encrypted WireGuard mesh network.

Children's Privacy

This application is not designed for or directed at children under 13. No data is knowingly collected from children.

Changes to This Policy

This policy may be updated as the application evolves. The "Last updated" date at the top reflects the most recent revision.

Contact

For questions about this privacy policy, contact the shop owner via the Etsy shop: MrsMillennialDesigns on Etsy.
